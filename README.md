## Tech Used

- Next.js v16.2 (React Framework)
- Tailwind CSS v4 (Utility Inline styling framework)
- React v19.2 (UI Library)
- Gsap v3.14 (JS Animation Library)
- Base UI v1.3 (Headless UI Components)
- Typescript v5 (JS Superset for forcing strict types and more)
- Shacn v4.1 (Designed Component Library)
- Bun.js v1.3 (Node.js and npm alternative)

## Installation

Staright forward setup

```bash
git clone https://github.com/17anirudh/anirudhh.git
cd anirudhh
bun install # or npm install
bun run dev # or npm run dev
```
